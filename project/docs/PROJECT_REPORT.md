# تقرير مشروع بوت تليجرام ديني شيعي

## ملخص تقني

| البيان | القيمة |
|---------|------|
| الأسم | Shia Religious Bot |
| التقنية | Python 3.11 + Aiogram v3 + APScheduler 3.10.4 + SQLite |
| إطار الصلاة | الجعفرية — الكوت (العراق) |
| التاريخ | يونيو 2026 |
| الحالة | كامل — جميع المراحل 11 منتهية |
| الحالة الحالية | البوت يعمل بـ 8 مهام مجدولة |

## المراحل والمهام المنجزة

### المرحلة 0 — Setup (البنية)
✅ إنشاء هيكل المشروع
✅ الـ `requirements.txt`
✅ الـ `config.py` + `.env`
✅ الـ SQLite initialization
✅ الـ Logging (خروج + ملف دوران)
✅ الـ `/start` يعمل

### المرحلة 1 — Core Bot
✅ القائمة الرئيسية (أزرار Inline)
✅ الأوامر: `/start`, `/help`, `/menu`, `/prayer`, `/event`
✅ حفظ المستخدم في قاعدة البيانات

### المرحلة 2 — Content Engine
✅ قراءة الـ JSON والـ cache
✅ منع التكرار بواسطة `sent_content`
✅ الـ `is_featured` والـ `send_score`
✅ الـ `split_message` للرسائل الطويلة
✅ الأنواع: حديث، حكمة، دعاء، مناجاة، زيارة، تعقيب

### المرحلة 3 — Subscriptions
✅ جدول `subscriptions` في قاعدة البيانات
✅ الـ `subscribe` / `unsubscribe`
✅ أزرار Inline تفاعلية (✅/❌)
✅ 4 اشتراكات: حديث، حكمة، دعاء، تعقيب

### المرحلة 4 — Prayer System
✅ حساب مواقيت الصلاة الجعفرية
✅ زواوي: فجر -16°، مغرب -4°، عشاء -14°
✅ عاصر الظل الشادو 1×
✅ المكان: الكوت (العراق)
✅ الاحداثي القادمة
✅ الـ `countdown`
✅ الـ `/prayer` مع الاحداثيات
✅ رسالة مواقيت مثبتة (مرسمة)

### المرحلة 5 — Pinned Prayer Message
✅ تحديث تلقائي كل دقيقة
✅ تثبيت الرسالة في القناة
✅ الـ `edit` إذا كانت مثبتة
✅ الـ `pin_chat_message`
✅ منع الإرسال المكرر

### المرحلة 6 — Adhan Notification
✅ الأذان التلقائي عند كل صلاة
✅ نص فقط واحد لكل صلاة يوميًا
✅ إرسال للقناة أو للمشتركين
✅ المناسبات الكبرى: استثناء بوقتي الاحتفال 2 دقيقة

### المرحلة 7 — Prayer Taqibat
✅ تعقيب الفجر بعد +3 دقائق
✅ تعقيب الظهر بعد +2 دقيقة
✅ تعقيب المغرب بعد +2 دقيقة
✅ الـ `split_message` للـ taqibat
✅ القناة أو للمشتركين

### المرحلة 8 — Daily Scheduler
| الوقت | المهمة | المشتركين |
|-------|--------|--------|
| 08:00 | حديث اليوم | مشتركي الحديث |
| 13:00 | حكمة يومية | مشتركي الحكمة |
| 18:00 | دعاء يومي | مشتركي الدعاء |
| 20:00 | مناجاة | مشتركي التعقيب |

### المرحلة 9 — Events System
✅ الـ `event_service.py` كامل
✅ التقويم الهجري بواسطة `hijri-converter`
✅ الـ `/event` يعرض المناسبة الحالية + القادمة
✅ فحص يومي تلقائي عند 07:00

### المرحلة 10 — Admin Panel
✅ الـ `IsAdmin` filter لحماية الأوامر
✅ الـ `/stats` — إحصائيات مفصّلة
✅ الـ `/broadcast` — إرسال جماعي
✅ الـ `/reload_json` — إعادة تحميل البيانات
✅ الـ `/test_hadith`, `/test_wisdom`, `/test_dua` — تجارب
✅ الـ `/test_prayer`, `/test_adhan`, `/test_taqibat` — تجارب
✅ الـ `/test_pinned`, `/test_event` — تجارب
✅ الـ `/jobs` — عرض المهام المجدولة

### المرحلة 11 — Optimization
✅ الـ Logging منظم مع ملف دوران
✅ منع الـالتكرار في الأذان والتعقيبات
✅ الـ Cache للـ JSON مع الـ `reload_all()`
✅ معالجة شاملة للأخطاء
✅ الـ `misfire_grace_time` في الـ APScheduler

| `daily_hadith` | 08:00 | حديث يومي |
| `daily_wisdom` | 13:00 | حكمة |
| `daily_dua` | 18:00 | دعاء |
| `daily_munajat` | 20:00 | مناجاة |
| `daily_event` | 07:00 | فحص المناسبات |
| `reschedule_prayers` | 00:01 | إعادة جدولة الصلوات |
| `adhan_<prayer>` | الوقت المحسوب | أذان |
| `taqibat_<prayer>` | بعد الأذان بدقائق | تعقيب |

وووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووووو
