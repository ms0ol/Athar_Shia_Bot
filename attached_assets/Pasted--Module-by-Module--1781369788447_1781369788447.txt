خطة بناء الكود Module by Module

أريد بناء مشروع البوت الديني الشيعي بطريقة تدريجية ومنظمة.

مهم جدًا:

* لا تبنِ كل شيء دفعة واحدة.
* لا تكتب كود Features مستقبلية الآن.
* لا تقفز بين Modules.
* لا تنشئ تعقيدًا غير ضروري.
* بعد كل Module يجب أن يكون المشروع قابلًا للتشغيل والاختبار.

اعمل بالترتيب الحرفي التالي.

⸻

قواعد المشروع

التقنية

Python

Telegram Framework:
Aiogram v3

Database:
SQLite فقط حالياً

normalized JSON files

المرحلة 5 — Pinned Prayer Message

الهدف:
رسالة مثبتة للقناة.

المطلوب:

رسالة ثابتة محدثة.

مثال:

🕌 الصلاة القادمة

المغرب
بعد: 01:23:11

القواعد:

* تحديث كل دقيقة.
* تعديل نفس الرسالة.
* لا يرسل رسالة جديدة كل مرة.

اختبار النجاح:

الرسالة تتحدث تلقائياً.
